To run this program, go to the src folder, open a command prompt, and enter
java Main
AND
enter your desired target steps file. In my example, I set this to outputStepsFile.txt
enter your desired target metrics file. In my example, I set this to outputMetricsFile.txt
enter the desired number of disks. The program will solve the puzzle from 1 disk up to the number of disks you specify.

The command line will print the current iteration which is being tested.
